# customer_sales_analysis
# 📊 Customer Behavior & Sales Analysis

## 📌 Project Overview

This project analyzes customer shopping behavior to extract actionable business insights.
The goal is to understand purchasing patterns, customer segments, and factors affecting revenue.

---

## 🎯 Objectives

* Analyze customer demographics and purchasing behavior
* Identify high-value customers
* Evaluate the impact of discounts and subscriptions
* Provide data-driven business recommendations

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy)
* MySQL
* Power BI
* Jupyter Notebook

---

## 📂 Dataset Description

The dataset includes:

* Customer demographics (Age, Gender)
* Purchase details (Category, Purchase Amount)
* Behavior (Frequency, Subscription Status)
* Ratings & Discounts

---

## 🔄 Data Processing

* Data cleaning and handling missing values
* Feature engineering (age groups, frequency conversion)
* Data transformation using SQL

---

## 📊 Key Insights

* Customers using discounts can still be high-value
* Clothing category generates the highest revenue
* Subscription status impacts overall spending
* Shipping type influences purchase behavior

---

## 💡 Business Recommendations

* Target high-value customers with personalized offers
* Improve low-performing categories
* Enhance subscription programs
* Optimize shipping strategies
